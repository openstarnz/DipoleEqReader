Installation
============

**dipoleq-reader** supports Python >= 3.8.

## Installing with `pip`

**dipoleq-reader** is available [on PyPI](https://pypi.org/project/dipoleq-reader/). Just run

```bash
pip install dipoleq-reader
```

## Installing from source

To install **dipoleq-reader** from source, first clone [the repository](https://github.com/jguterl/DipoleEqReader):

```bash
git clone https://github.com/jguterl/DipoleEqReader.git
cd DipoleEqReader
```

Then run

```bash
pip install -e .
```
